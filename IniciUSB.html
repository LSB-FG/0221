Què és un USB d'arrencada?

Un USB d'arrencada és una memòria USB preparada per iniciar un sistema operatiu o eines de reparació sense necessitat d'accedir al disc dur de l'ordinador. Això és essencial per a diagnòstic i reparació de problemes, recuperació de dades o instal·lació de sistemes operatius.

Eines Imprescindibles

Rufus

Eina lliure i de codi obert per crear USB d'arrencada amb imatges ISO. Ràpid i eficient per a instal·lacions de Windows o Linux.

Hiren's BootCD

Col·lecció d'eines de reparació incloent antivirus, recuperació de dades, clonació de discs, editor de registre i moltes més utilitats.

Visita el web oficial

Ultimate Boot CD

Col·lecció d'eines de diagnòstic i reparació per a maquinari. Inclou utilitats per proves de RAM, CPU, disc dur i components del sistema.

Com crear un USB d'arrencada amb Windows

1

Descarrega la ISO de Windows

Descàrrega la imatge ISO del Windows que vulguis instal·lar des de la pàgina oficial de Microsoft.

2

Prepara el USB

Connecta una memòria USB amb mínim 8GB d'espai (tot el contingut s'esborrarà). Assegura't que no hi ha dades importants.

3

Utilitza Rufus

Executa Rufus, selecciona el dispositiu USB, tria la imatge ISO descarregada i prem "Començar". Espera que finalitzi el procés.

Assegura't que l'opció "Crear un dispositiu d'arrencada" està seleccionada i el "Sistema de destí" és "UEFI" o "BIOS" segons necessitis.

4

Arrenca des de l'USB

Reinicia l'ordinador i accedeix al menú d'arrencada (normalment prement F12, ESC o la tecla indicada a la pantalla inicial). Selecciona el USB com a dispositiu d'arrencada.

Casos Pràctics amb USB d'Arrencada

Desinfecció avançada de virus

Quan un virus impedeix l'arrencada normal del sistema, un USB amb Hiren's BootCD permet arrencar en un entorn segur i utilitzar antivirus com Kaspersky TDSSKiller o Malwarebytes per eliminar amenaces persistents.

Recuperació de particions

Amb TestDisk (inclòs a Hiren i Ultimate Boot CD) es poden recuperar particions esborrades accidentalment o corregir taules de particions danyades, salvant dades valuoses sense necessitat d'instal·lar res al disc dur.

Recuperació de contrasenyes

Utilitzant eines com NTPasswd o Offline NT Password & Registry Editor es poden canviar contrasenyes d'administrador perdudes en sistemes Windows sense necessitat de conèixer l'antiga contrasenya.

Clonació de discs

Amb Clonezilla o Macrium Reflect pot fer còpies d'imatge completes d'un disc dur per migrar a una unitat SSD més ràpida o crear còpies de seguretat abans de resoldre problemes complexes.

Consells per a Tècnics

Seguretat

Sempre treballa amb còpies de seguretat abans de fer canvis dràstics. Moltes eines de recuperació poden sobreescriure dades per error.

Kit bàsic

Té sempre preparats diversos USB amb diferents eines: un per a Windows, un amb Hiren's BootCD i un altre amb distribucions Linux d'emergència com SystemRescueCd.

Actualització

Actualitza periòdicament els USB d'arrencada amb les últimes versions de les eines, especialment els antivirus i eines de diagnòstic per ser efectiu amb els últims malware i dispositius.
